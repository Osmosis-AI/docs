This guide walks you through setting up a repository to sync with Osmosis AI platform.

<Tip>
For a complete working example, see the [official example repository](https://github.com/Osmosis-AI/osmosis-git-sync-example).
</Tip>

## Prerequisites

- GitHub account
- Python 3.10 or higher
- Git installed locally
- Osmosis AI account

## Step 1: Set Up Repository

Create the required directory structure and add dependencies. See [Folder Structure](/git-sync/folder-structure) for full details.

```bash
mkdir -p my-osmosis-repo/{mcp/{server,tools,test},reward_fn,reward_rubric}
cd my-osmosis-repo
```

Create `pyproject.toml` in the repository root:

```toml
[project]
name = "my-osmosis-repo"
version = "0.1.0"
requires-python = ">=3.10"
dependencies = [
    "osmosis-ai>=0.1.0",
    "fastmcp>=0.1.0",
]

[build-system]
requires = ["setuptools>=61.0"]
build-backend = "setuptools.build_meta"
```

## Step 3: Test Locally

```bash
# Install in development mode
pip install -e .

# Test MCP server
python mcp/main.py

# Test reward function
python -c "from reward_fn.compute_reward import numbers_match_reward; print(numbers_match_reward('#### 42', '42'))"
```

## Step 4: Push to GitHub

```bash
git init
git add .
git commit -m "Initial Osmosis sync setup"
git remote add origin https://github.com/your-username/my-osmosis-repo.git
git push -u origin main
```

## Step 5: Connect to Osmosis Platform

Follow the [GitHub Integration](/platform/github-integration) guide to install the Osmosis GitHub App and connect your repository. Once connected, Osmosis automatically syncs on every push.

To verify:

1. Check the **Git Sync** dashboard for sync status
2. Navigate to **MCP Tools**, **Reward Functions**, and **Reward Rubrics** to confirm your components appear
