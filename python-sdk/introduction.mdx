# Osmosis AI Python SDK

The `osmosis-ai` Python SDK provides tools for evaluating LLM outputs and building custom agents for reinforcement learning training on the [Osmosis Platform](https://platform.osmosis.ai).

<Note>
**Prerequisite**: You need an [Osmosis Platform account](/platform/quickstart) to use the SDK with the training platform. Install and authenticate with `pip install osmosis-ai && osmosis login`.
</Note>

## Quick Install

```bash
pip install osmosis-ai
```

For Remote Rollout servers:
```bash
pip install "osmosis-ai[server]"
```
