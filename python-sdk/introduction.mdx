# Osmosis Python SDK

The Osmosis Python SDK provides a convenient way to interact with the Osmosis Agent Improvement API from your Python applications.

## Installation

Install the SDK using pip:

```bash
pip install osmosis-sdk
```

## Quick Start

```python
from osmosis import OsmosisClient

# Initialize the client with your API key
client = OsmosisClient(api_key="your-api-key")

# Enhance a task with relevant knowledge
response = client.enhance_task(
    tenant_id="your-tenant-id",
    query="How do I reset a user's password?",
    context={"user_role": "admin"}
)

print(response.enhanced_query)
```
