The Osmosis API provides a simple API for building agentic workflows. To start using:

<Steps>
  <Step title="Create an API Key">
    Create an API key in your [workspace settings](https://platform.osmosis.ai/settings/workspace/api-keys) on the Osmosis platform.
  </Step>
  <Step title="Authentication Setup">
    Follow our [authentication guide](/api-reference/authentication) to set up your API key
  </Step>
  <Step title="Make Your First Call">
    Try our [API endpoints](/api-reference/enhance-task) with the provided examples
  </Step>
</Steps>
